# Expense counter

### Описание
Позволяет выбрать категорию и посчитать расходы. При превышении лимита текст статуса изменяется на красный цвет и пишет: всё плохо. Лимит можно менять.

### Языки и техногии
HTML, CSS, JavaScript.
